# Dimitri.app - Personal Portfolio

Professional portfolio website for Dimitrios Pantzos, showcasing experience as a foodservice executive and entrepreneur.

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Icons**: Lucide React
- **Deployment**: Vercel

## Features

- Single-page modern design
- Dark theme
- Mobile-responsive
- Professional experience timeline
- Contact information and CTAs

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This site is designed to be deployed on Vercel:

1. Connect your GitHub repository to Vercel
2. Configure custom domain: dimitri.app
3. Deploy automatically on push to main branch
